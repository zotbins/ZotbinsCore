### Applying doxygen to Your Project

You can run doxygen locally using the following instructions. 

### 1. Install Doxygen GUI Frontend

Instructions on this Google Doc (Zotbins drive -> Embedded Systems -> Embedded Systems Team -> Documentation -> Doxygen documentation How-to): https://docs.google.com/document/d/1ouunmtmhAlxnZCdb7q7WhkATDgulp6o9y7Jy3oYrSqY/edit?tab=t.0#heading=h.jxy160bbpn8l


### 2. Run the following commands in the terminal
doxygen .\.doxygen\Doxyfile  <- this essentially parse through the different files using the config settings in the Doxyfile
start .\docs\html\index.html <- this autogenerates a website that should pop up in your default browser

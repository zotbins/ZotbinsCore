Detailed Description

Servo control API for ESP32.

Provides initialization and simple position control for a TowerPro MG996R servo using PWM. All functions return ESP-IDF style error codes.

◆ init_servo()

esp_err_t init_servo ( void )

Initialize the servo driver.

Sets up the PWM timer/channel and any required GPIO for the servo signal.

Returns
  • ESP_OK on success
  • ESP_ERR_INVALID_STATE if already initialized
  • ESP_FAIL (or another esp_err_t) on hardware/config error

◆ servo_set_angle()

esp_err_t servo_set_angle ( float degrees)

Set the servo angle.

Converts degrees to the proper PWM pulse and updates the servo output.

Parameters
degreesTarget angle in degrees (e.g., 0–180). Values outside the supported range will be clamped or return an error depending on the implementation.
Returns
  • ESP_OK on success
  • ESP_ERR_INVALID_ARG if degrees is out of range
  • ESP_ERR_INVALID_STATE if the driver is not initialized
